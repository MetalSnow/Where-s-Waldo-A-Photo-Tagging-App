# Where-s-Waldo-A-Photo-Tagging-App

A full-stack Where’s Waldo inspired game built with **React** and **Node.js**.

Users must find hidden characters in large scene images. The app validates selections with the backend, tracks completion time, and displays a leaderboard.

---

## 🚀 Features

- Click-to-select targeting box
- Backend validation of character positions
- Timer tracking (stored in seconds)
- Leaderboard for multiple scenes
- Current user highlight
- Editable username
- Scrollable styled score tables

---

## 🛠 Tech Stack

**Frontend**

- React
- React Router
- Custom Hooks
- CSS

**Backend**

- Node.js
- Express
- REST API
